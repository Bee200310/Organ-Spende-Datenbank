# Organ-Spende-Datenbank
************************
In diesem Projekt Findest du eine Datenbank Software zum Spenden und Suchen von Passen den Organen.
******************************************
Pflichtenheft Projekt Organspende/Datenbank 

Gruppe: Slay Bitches 			
Klasse: IAH11 
Mitglieder: Belkiz, Julia								 
Projektziel (2-3 Sätze) 

Wir wollen ein Organspede Programm/Datenbank zusammenstellen. Wo man nach einen gespendeten organ suchen kann und das dann Anfragen kann. Außerdem wollen wir die funktion haben das sich leute als organspender eintragen können. Diese programm ist für Ärzte, Krankenhäuser und Privatpersonen verwendet werden. 

Muss-Kriterien 

1.1. Muss einen Lade Screen haben. 

1.2. Muss eine Startseite mit Search bar haben um nach gespendeten organen zu suchen. 

1.3. Man muss bei der suche einen Blutgruppe angeben Können. 

1.4. Wenn ein Match(Passendes Organ zum tranzplantieren) gefunden wird muss eine Meldung für den User erscheinen. 

1.5. Wenn kein Match gefunden wird muss eine Meldung für den User erscheinen. 

1.6. Verstorbene müssen nach Sterbe datum sortiert werden. 

1.7. Verstorbene müssen mit(TODID,Blutgruppe,Geburtsdatum,Alter,Bezeichnung des  
Organs (Welches Organ)gespeichert werden.    

1.8.Für Menschen die sich als Spender eintragen wollen müssen(Name,Vorname,Geburtsdatum,Welche Organe gespendetwerden) gespeichert Werden. 

1.9. Es Wird Die Entität Verstorbene mit den Attributen(ID,Blutgruppe,Geburtsdatum,Alter,Bezeichnung des Organs)erstellt.

Kann-Kriterien 
**************

Z.1.Info page Warum Organspenden so wichtig sind. 

Z.2.Ein page wo mann geschichten von Transplantierten menschen nachlesen kann. 

Z.3.Eine Kontak page bei Problemen.
